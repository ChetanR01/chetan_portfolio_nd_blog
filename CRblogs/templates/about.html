About This Website

This is a Blog Website containing blogs of different categories and domains. I have developed this website to practice my Django skills. The website front-end template in downloaded from TemplateMo. I did some changed in template required for this website. And added all Backend related stuff using Python-Django. This website provides all the features listed below. I used the data for the blogs from Wikipedia in starting. I will try to add my own blog posts on this website in near future.
